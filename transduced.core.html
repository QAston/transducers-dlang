enum  isExtendedOutputRange(O, ElementType);

An extended output range is a std.range.OutputRange with 2 additional capabilities: early termination using O.isDone() and buffering using O.flush() which flushes the range buffer.

All extended output ranges are also regular output ranges. All Putters are also extended output ranges.

Examples
static assert(isExtendedOutputRange!(Putter!(int, int[]), int));

enum  isTransducer(T, InputType);

Is true when T is a transducer type which can process InputType input.

Transducer is an object with wrap!InputType method taking a (possibly already Decorated) Putter object to decorate and returning a DecoratedPutter (Putter wrapped in instance). The returned DecoratedPutter accepts InputType input. Transducer also has OutputType(InputType) alias which defines what type will be passed to DecoratedPutter on given input.

Examples
// a dummy transducer object
struct Tducer
{
    auto wrap(InType, Decorated)(Decorated putter) if (isPutter!Decorated)
    {
        struct PutterDecorator
        {
            mixin PutterDecoratorMixin!(Decorated, InType);
            this(Decorated putter)
            {
                this.putter = putter;
            }

            void put(InputType p)
            {
                std.range.put(putter, p);
            }
        }

        return PutterDecorator(putter);
    }

    alias OutputType(InputType) = InputType;
}

static assert(isTransducer!(Tducer, int));

template  PutterDecoratorMixin(Decorated, InType)

Mixin used to implement common code for all putter decorators.

Specific putter decorators provide additional capabilities to Putter by wrapping it and forwarding calls to the Decorated putter object. InType - type for this decorator to take in PutterDecorator.put.

For more information on how decorators work google "Decorator design pattern".

bool  isDone();

Early termination. Returns true when there's a guarantee that this Putter won't std.range.put new data to PutterDecorator.to for any future Putter.put calls, so they can be skipped.

This check can be used for performance (in loops - to reduce calls which won't result in any output) and to provide early termination semantics (i.e. when working with infinite ranges).

By default always returns false. If the wrapped OutputRange is an ExtendedOutputRange, forwards call to ExtendedOutputRange. isDone.

void  flush();

Flushes any input buffered so far by PutterDecorators using Putter.put. Can be called multiple times, or even not called at all.

Usually called when finished Putter.putting.

If the wrapped OutputRange is an ExtendedOutputRange, forwards call to ExtendedOutputRange. flush.
