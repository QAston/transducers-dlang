enum  isCopyable(T);

Returns true if type T is copyable.

auto  own(T)(ref T src);

Takes ownership from provided lvalue into an rvalue. Then ownership of the rvalue can be transfered freely according to the rules of D language.

For copyable types this is simply returns the value. For non-copyable types returns result of calling std.algorithm.move, which copies from given src into an rvalue, and resets src using init.

struct  PutterBuffer(T, Allocator);

A queue like buffer that works in 2 phases: filling element by element using put and removing element by element using removeFront() until it's empty.

Non-copyable. Owns allocated memory and frees it upon destruction.
